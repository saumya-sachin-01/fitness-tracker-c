# Personal Health & Fitness Tracker (C Project)

## About the Project

This is a console-based application developed in C to track daily health and fitness activities. The program allows users to record their routine habits such as sleep, meals, exercise, and water intake. The main objective of this project is to demonstrate the use of file handling and basic programming concepts in a practical scenario.

---

## Features

* Create user profile (Name, Age, Gender)
* Add and manage fitness goals
* Record daily activities
* Provide basic feedback based on inputs
* Save records into a file
* View previously stored data

---

## Concepts Used

* Basic C programming
* Conditional statements (if-else)
* Looping (menu-driven approach)
* File handling (read and write operations)
* String handling

---

## How to Run

1. Compile the program:

```bash
gcc fitness.c -o fitness
```

2. Run the executable:

```bash
./fitness
```

---

## Data Storage

All user records are stored in a file named:

```id="nprm0v"
fitness_record.txt
```

This ensures that the data remains available even after the program is closed.

---

## Objective

The purpose of this project is to apply theoretical knowledge of C programming to a real-world use case. It helps in understanding how user input can be processed, analyzed, and stored efficiently.
